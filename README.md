# SmartPredict | Predictive Maintenance Agent for Smart Manufacturing

SmartPredict is an Industry 4.0 intelligent machinery diagnostics and predictive maintenance dashboard. It simulates an active manufacturing plant featuring 20 machines (CNC Milling Centers, Robotic Arms, and Hydraulic Pumps), analyzing live sensor streams (vibration, temperature, RPM, current draw) using machine learning algorithms to predict mechanical downtime and schedule preventive maintenance tickets automatically.

---

## 🌟 Key Features

1. **Digital Twin Virtualization**: Real-time animated SVGs of CNC mills, robotic arms, and hydraulic pumps. The virtualization components spin, cool, and highlight diagnostic heat hotspots dynamically based on incoming sensor readings.
2. **AI Predictive Analytics (ML Pipeline)**: An automated machine learning pipeline executing feature scaling, training an Isolation Forest for anomaly detection, a Random Forest Classifier for failure probability risk, and a Random Forest Regressor for Remaining Useful Life (RUL) estimation.
3. **Explainable AI (XAI)**: Displays SHAP-value approximations illustrating which sensor values (such as a spindle current spike or high motor vibration) are pushing the machine toward failure.
4. **Interactive Fault Injector**: Allows engineers to stress-test the AI agent by injecting live anomalies, including bearing wear, temperature spikes, coolant pressure leaks, and motor electrical overloads.
5. **AI Chat Copilot**: An intelligent Chat Assistant (using local context RAG, with optional upgrade to OpenAI/Gemini APIs) that answers operator queries (e.g., *"Why is Machine 1 at risk of failing?"*).
6. **Maintenance Scheduler**: Automated ticketing schedules generated by AI predictions, featuring priority levels, estimated downtimes, technician assignments, and an interactive calendar.
7. **Predictive Stocking Inventory**: Correlates machine RUL countdowns directly with spare-parts stock levels and reorder lead times, flagging urgent procurement tickets before machines break down.
8. **Executive Reports**: Generates downloadable PDF summaries and Excel operations spreadsheets mapping plant diagnostics.

## 🛠️ Technology Stack

* **Frontend**: React.js, Tailwind CSS (Design System), Recharts (Visualizations & Graphs), Lucide (Icons)
* **Backend**: FastAPI (Python), WebSockets (Real-time telemetry stream), SQLite (Development database / SQLAlchemy), PostgreSQL (Production ready)
* **ML/Data Science**: Pandas, NumPy, Scikit-learn (Random Forest, Isolation Forest)
* **Orchestration**: Docker, Docker Compose

## 🚀 Quick Start (Docker Compose)

The entire project is pre-configured to build and execute locally with a single command. 

Ensure you have [Docker](https://www.docker.com/) installed, then run:

```bash
docker-compose up --build
```

* **Frontend Dashboard**: Open [http://localhost:3000](http://localhost:3000)
* **Backend Swagger API Docs**: Open [http://localhost:8000/docs](http://localhost:8000/docs)

---

## 💻 Manual Local Installation (Without Docker)

### 1. Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create a virtual environment and activate it:
   ```bash
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Initialize the SQLite database and seed initial metrics:
   ```bash
   python app/db/init_db.py
   ```
5. Run the FastAPI development server:
   ```bash
   uvicorn main:app --reload
   ```

### 2. Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd ../frontend
   ```
2. Install npm modules:
   ```bash
   npm install
   ```
3. Launch the Vite server:
   ```bash
   npm run dev
   ```
4. Access the portal at [http://localhost:3000](http://localhost:3000)
